# Telnyx Ctpress 


## 1. Install 
in terminal enter
```
git clone https://github.com/ValeriiMandryka/playwrite.git
npm ci 
```
## 2. Run tests
```
npx cypress open   (it will open cypress UI where you can run tests)  
```
OR

## Run tests in head mode
```
npx cypress run (it will run tests in your IDE)
```
